# Quick Reference: Feedback Loop Bridge ⚡

## 🚀 Comandos Rápidos

- `python run.py learn --from-logs`: Analizar logs locales y extraer lecciones.
- `python run.py memory-sync`: Sincronizar memoria YAML con BrainDB.

## 🛠️ Atajos y Constantes

- **Memory Path**: `.ai/memory/lessons_learned.yaml`
- **Logs Dir**: `logs/`

## 💡 Tips de Expertos

- No todas las excepciones son lecciones; filtra el ruido.
- Las lecciones más valiosas vienen de los fallos en tests de integración.
- Mantén las lecciones cortas y accionables.
