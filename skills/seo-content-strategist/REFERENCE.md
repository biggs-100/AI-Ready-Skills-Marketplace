# SEO Strategy Reference 🚀⚡

## 🚀 Checklist de Visibilidad (10 Puntos)

1. [ ] **Unique Titles**: ¿Cada página tiene un `<title>` único y menor a 60 caracteres?
2. [ ] **Meta Descriptions**: ¿Hay una descripción atractiva menor a 155 caracteres?
3. [ ] **H1 Rule**: ¿Solo hay un `<h1>` por página y contiene la palabra clave principal?
4. [ ] **Alt text**: ¿Todas las imágenes tienen el atributo `alt` coherente?
5. [ ] **JSON-LD**: ¿Se ha inyectado el script de datos estructurados para esta página?
6. [ ] **Canonical**: ¿Existe la etiqueta `rel="canonical"` para evitar contenido duplicado?
7. [ ] **Social Metas**: ¿Están presentes `og:image` y `og:title`?
8. [ ] **Internal Linking**: ¿La página está enlazada desde otros puntos de la app?
9. [ ] **Mobile Friendly**: ¿El diseño es totalmente responsivo y legible en móvil?
10. [ ] **Favicon & Manifest**: ¿Están configurados correctamente para la experiencia de usuario?

## 🛠️ Herramientas de Validación

- `Google Search Console`: Para verificar el rastreo.
- `Lighthouse SEO`: Para auditoria rápida desde el navegador.
- `Rich Results Test`: Para validar el JSON-LD.

## 💡 Tip: Contenido

Evita el "Keyword Stuffing". Escribe para humanos, pero estructura para IAs (Google es una IA). Usa sinónimos y variaciones semánticas de tu tema principal.
