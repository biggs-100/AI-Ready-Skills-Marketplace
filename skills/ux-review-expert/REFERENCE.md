# UX Reference (UX Pro Max v2.0) 🎨⚡

## 🚀 Tokens Rápidos (CSS Variables)

- **Fondo**: `var(--background)`
- **Principal**: `var(--primary)` / `var(--primary-foreground)`
- **Éxito**: `var(--success)`
- **Bordes**: `var(--border)`
- **Radio**: `var(--radius)` (aprox. 12px)

## 🛠️ Clases de Utilidad Premium

- `.glass`: Efecto cristal esmerilado con blur.
- `.gradient-text`: Texto degradado (Primary -> Success).
- `.professional-btn`: Botón con elevación y sombra táctica.
- `.bento-card`: Contenedor con hover suave y elevación.

## 💡 Regla de Oro

Si quieres un color que no sea `primary`, usa siempre `hsl()` con una saturación entre el 60-80% para mantener el look vibrante pero profesional.
